# Word Scramble Game 🎮

This is a simple C++ Word Scramble Game built using SFML.

## Features
- Random scrambled words
- Timer-based gameplay
- Multiple levels
- User-friendly interface

## Technologies Used
- C++
- SFML Library

## How to Run
1. Compile the code using g++
2. Run the executable file
